# 内置插件

内置插件是 Nekro Agent 随系统提供的基础能力。它们覆盖消息交互、图片查看、定时任务、搜索、语音、掷骰、群管理等常见场景。

大多数用户可以先从内置插件开始，不需要一上来安装很多社区插件。

## 在哪里启用

进入 WebUI 的「插件管理」页面，可以查看、启用、停用和配置插件。

建议按下面的顺序使用：

1. 先确认基础插件保持启用
2. 根据需要启用搜索、语音、定时器等扩展插件
3. 配置插件需要的密钥或参数
4. 在频道中测试插件是否能正常工作

::: tip 推荐
刚开始使用时，建议只启用确实需要的插件。插件越少，AI 可选能力越清晰，排查问题也越简单。
:::

## 常用内置插件

| 插件 | 可以做什么 | 适合场景 |
|---|---|---|
| 基础交互插件 | 发送文本、图片、文件，查看部分数据内容 | 日常聊天、文件发送、基础交互 |
| 图片查看 | 帮助 AI 查看和理解图片内容 | 图片识别、截图分析、表情包理解 |
| 定时器工具集 | 创建提醒、一次性定时任务和周期任务 | 定时提醒、每日播报、周期巡检 |
| Google 搜索工具 | 获取实时搜索结果 | 查询新闻、资料、网页信息 |
| AI 语音插件 | 将文本转换为 AI 语音 | 语音回复、角色语音输出 |
| 掷骰姬 | 掷骰检定和随机选择 | TRPG、随机事件、群聊娱乐 |
| 风纪委员 | 协助群管理和禁言 | 群聊管理、违规处理 |
| 笔记插件 | 记录和读取简单笔记 | 临时记录、轻量信息保存 |
| 白板插件 | 维护可反复修改的共享内容 | 群内公告草稿、协作记录 |
| 历史穿越 | 回看或整理历史消息 | 追溯上下文、总结讨论 |
| 状态插件 | 查看系统或会话状态 | 排查问题、确认运行状态 |
| 动态导入工具 | 让 AI 在需要时补充部分 Python 能力 | 数据处理、网页解析、脚本任务 |

## 平台相关插件

部分内置插件只在特定平台或适配器场景下更有用。

| 插件 | 适合场景 |
|---|---|
| Bilibili 直播工具插件 | Bilibili 直播适配器消息发送、Live2D 表情、音效和动画控制 |
| Minecraft 工具插件 | Minecraft 相关服务器或玩家交互场景 |
| 邮箱工具插件 | 邮件发送、邮件相关自动化任务 |
| 群荣誉插件 | 支持群荣誉等群聊信息相关能力 |

如果您没有使用对应平台，可以保持这些插件停用。

## 需要额外配置的插件

有些插件启用后还需要填写配置才能正常使用。

常见例子：

| 插件 | 需要准备什么 |
|---|---|
| Google 搜索工具 | Google 搜索 API Key 和 CX |
| AI 语音插件 | 语音服务地址、角色或相关服务配置 |
| 邮箱工具插件 | 邮箱账号、授权码、SMTP/IMAP 等信息 |
| 风纪委员 | 管理权限、管理频道或禁言上限等配置 |
| Bilibili 直播工具插件 | 直播适配器相关配置 |

具体配置项以插件管理页面显示为准。

## 使用建议

### 保持基础能力稳定

基础交互、图片查看、状态等插件通常是日常使用中比较基础的能力。除非您明确知道影响，否则不建议随意停用。

### 不要重复启用同类能力

如果多个插件都能完成相似任务，AI 可能会难以选择。建议保留一个稳定、常用的插件即可。

### 正式环境谨慎开启高权限能力

涉及群管理、服务器、外部服务调用的插件，建议先确认权限范围和使用场景，再在正式频道启用。
