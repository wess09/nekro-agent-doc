::: warning 高风险
不建议使用主力微信账号接入任何机器人框架，包括 WeChatPadPro。存在账号异常、限制功能甚至封号的风险。

- API 运行环境和手机所在地差太远，容易触发风控
- 云服务器异地登录风险更高
- 长时间无人手动操作的账号做高频自动化，也容易异常
