# 隐私政策

最后更新日期：2025年12月16日

欢迎您访问 NekroAgent 文档网站（以下简称"本网站"）。我们非常重视您的隐私保护，并致力于透明地处理您的个人信息。本隐私政策旨在帮助您了解我们如何收集、使用、存储和保护您的信息。

## 1. 信息收集

### 1.1 自动收集的信息
当您访问本网站时，我们可能会自动收集以下信息：
- Cookie 数据：用于改善您的浏览体验

### 1.2 Cookie 使用
我们使用 Cookie 和类似技术来：
- 记住您的语言选项 (因为 VitePress 内置的多语言支持使用 Cookie)
- 记住您的深色或浅色模式设定

您可以通过浏览器内置的功能来管理Cookie设置以拒绝Cookie。

## 2. 信息使用

我们收集的信息用于以下目的：
- 个性化您的用户体验

## 3. 信息共享

我们不会将您的个人信息出售给第三方。在以下情况下，我们可能会共享信息：

- 法律法规要求

## 联系我们

如果您对本隐私政策有任何疑问，请通过以下方式联系我们：
- GitHub Issues：[点击此处](https://github.com/KroMiose/nekro-agent-doc/issues/new?template=issue_template.yml)
