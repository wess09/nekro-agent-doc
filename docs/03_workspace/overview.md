# 工作区总览

工作区是 Nekro Agent 最核心的能力组织方式。

## 功能列表

| 功能 | 可以做什么 |
|---|---|
| Claude Code 沙盒 | 执行代码、处理文件、生成报告、完成复杂任务 |
| 知识库 | 上传文档资料，让 AI 基于资料回答问题 |
| 记忆系统 | 记录长期背景、偏好和项目状态，减少重复说明 |
| 资源中心 | 管理服务器、数据库、令牌等资源，并按需挂载给工作区 |
| MCP 服务 | 连接外部工具和服务，扩展工作区能力 |
| 工作区定时器 | 定时提醒、周期巡检、自动执行固定任务 |
| 提示词与技能 | 为工作区设置长期要求和专用能力 |
| 通讯与扩展 | 查看工作区关联记录，管理扩展能力和工作区配置 |

::: tip 推荐使用流程
建议按 **部署 → 插件 → 工作区总览** 的顺序阅读和配置：先完成基础部署，再了解插件能力，最后创建工作区承载长期任务。
:::

当您需要 AI 不只是聊天，而是真正帮您**做事**——整理资料、分析数据、写报告、跟进项目、定期巡检——工作区就是承载这一切的地方。

一个工作区把 **AI 执行环境（Claude Code 沙盒）、知识库、长期记忆、外部工具（MCP）和定时任务** 组织在一起，围绕同一个目标持续工作。频道里的对话不再是一次性的问答，而是一个持续推进的项目空间。

![工作区列表页与创建工作区入口](/assets/workspace/overview/workspace_list_and_create.png)

## 工作区能带来什么

### 从「聊天」到「做事」

没有工作区时，AI 只能在对话里回答问题。有了工作区，AI 可以：

- 在 Claude Code 沙盒中编写和执行代码，处理文件、生成报告
- 通过 MCP 服务连接外部工具和数据源
- 自动记住之前聊过什么、做过什么
- 按照您设定的时间表，自动执行周期性任务

### 互不干扰的任务空间

每个工作区是独立的。不同项目的知识、记忆和工具配置互不干扰。您可以同时运行一个「数据分析」工作区和一个「内容创作」工作区，它们各自保持自己的上下文和能力。

### 越用越好用

工作区里的记忆系统会持续从对话和任务中提取关键信息。用得越久，AI 对您的项目理解越深，重复交代背景的频率越低。

## 工作区的能力组成

| 能力 | 作用 | 是否依赖 CC 沙盒 |
|---|---|---|
| **Claude Code 沙盒** | 代码执行、文件处理、复杂任务的核心引擎 | — |
| **知识库** | 上传文档资料，AI 自动检索引用 | 否 |
| **记忆系统** | 自动提取和保留长期背景，越用越懂您 | 否 |
| **资源中心** | 管理可复用的服务器、数据库、令牌等资源，并按需挂载到工作区 | 是 |
| **定时器** | 让 AI 按时自动执行任务 | 否 |
| **CC 模型组** | 为 Claude Code 沙盒独立配置模型 | — |
| **提示词与技能** | 工作区级别的行为定制和能力配置 | 否 |

::: tip 推荐
建议为每个正式使用的频道创建独立的工作区。即使暂时不需要 Claude Code 沙盒，知识库和记忆系统也能显著提升 AI 的表现。
:::

## 工作区详情页

进入某个工作区后，可以通过不同 Tab 管理该工作区的能力：

| Tab | 用途 |
|---|---|
| 总览 | 查看工作区基本状态和主要能力入口 |
| 沙盒 | 管理 Claude Code 沙盒和任务执行环境 |
| 通讯 | 查看工作区关联频道、对话和任务记录 |
| 记忆 | 管理工作区长期记忆 |
| 知识库 | 上传、绑定和检索工作区资料 |
| 扩展 | 管理与工作区相关的扩展能力 |
| MCP | 配置工作区可用的 MCP 服务 |
| 资源 | 挂载服务器、数据库、令牌等资源 |
| 提示词 | 设置工作区长期要求和行为偏好 |
| 配置 | 调整工作区基础参数和运行选项 |

首次使用时，不需要一次性配置所有 Tab。建议先完成基础信息、知识库或记忆，再按任务需要启用沙盒、MCP、资源和技能。

## 典型使用场景

### 项目管理助手

创建一个工作区，上传项目文档到知识库，绑定项目讨论频道。AI 会记住项目进展，您可以让它定期生成进度报告、追踪待办事项。

### 数据分析工作台

创建一个工作区，启用 Claude Code 沙盒。直接把数据文件丢给 AI，它可以在沙盒中用 Python 分析数据、生成图表，并把结果文件发回给您。

### 自动化运维

创建一个工作区，配置 MCP 服务连接您的监控系统，设置定时器每小时检查一次。AI 会自动巡检并在发现异常时通知您。

### 知识管理中心

创建一个工作区，持续上传团队文档到知识库。任何成员在频道中提问，AI 都能基于知识库给出准确回答。

### 长期陪伴角色

创建一个工作区，启用记忆系统。AI 会记住您们之间的对话、您的偏好和习惯，对话体验远超普通聊天。
