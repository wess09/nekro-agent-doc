# 会话独立人设

会话独立人设的作用，是让不同会话可以使用不同的人设。

这样做最直接的好处是，不同场景不用共用同一个说话风格和行为设定。

## 应用场景

1. **多角色对话**：同时与多个不同 AI 角色进行对话，每个会话保持独立人格
2. **领域区分**：为娱乐群组和生产力群组设置不同人设，生产娱乐两不误！
3. **测试与对比**：并行测试不同人设的响应效果

## 什么时候适合用

下面这些情况通常都适合开启会话独立人设：

- 不同群聊承担的用途完全不同
- 您希望工作群和娱乐群的人设分开
- 您想测试多个角色设定的效果

## 如何设置

![人设管理页面](/assets/advanced/session_persona/persona_management.png)

1. 先进入 `人设管理`
2. 新建或准备好要使用的人设
3. 再到目标会话里选择对应人设
4. 保存后开始测试

![会话人设选择](/assets/advanced/session_persona/session_persona_select.png)

## 使用建议

- 不要给所有会话都配不同人设，维护成本会很高
- 先从差异最明显的几个会话开始
- 如果只是小范围测试，先在低风险频道试用
