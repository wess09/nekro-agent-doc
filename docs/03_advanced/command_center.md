# 命令中心

命令中心是统一管理和执行命令的入口。

如果您平时需要查看某条命令有没有启用、在哪个频道可用、权限是什么，或者想直接在 WebUI 里试跑一条命令，基本都会用到这个页面。

## 这个页面能做什么

在命令中心里，通常可以完成这些事情：

- 查看当前有哪些命令
- 按分类、来源和权限筛选命令
- 查看某条命令的说明、别名和参数
- 调整命令是否启用
- 调整命令权限
- 针对单个频道做覆盖设置
- 在 WebUI 中直接执行命令
- 实时查看命令输出

![命令中心页面](/assets/advanced/command_center/command_center_overview.png)

## 管理范围怎么理解

命令中心通常分两种范围：

- `系统默认`
- `指定频道`

如果您在系统默认下修改，影响的是全局默认行为。  
如果您切到指定频道修改，影响的是这个频道自己的覆盖设置。

## 什么时候用系统默认

下面这些情况更适合在系统默认里处理：

- 您想全局启用或停用一条命令
- 您想统一修改某条命令的默认权限
- 您只想先看整个系统有哪些命令

## 什么时候用指定频道

下面这些情况更适合切到指定频道：

- 某条命令只想在一个频道里开放
- 某个频道想单独禁用一条命令
- 您要排查“为什么这个频道里命令不能用”

## 命令详情里通常看什么

点开某条命令后，最值得先看的通常是：

- 命令说明
- 用法
- 别名
- 参数
- 生效权限
- 来源

如果是插件命令，也可以顺便看它来自哪个插件。

## WebUI 里直接执行命令有什么用

如果您不想专门回到聊天平台测试，也可以直接在命令中心里选频道后执行命令。

这更适合：

- 快速验证命令是否可用
- 查看命令输出
- 排查参数问题
- 复现某个频道里的命令行为

![命令中心执行区与输出区](/assets/advanced/command_center/command_execute_and_output.png)

## 使用建议

- 先看命令详情，再改开关和权限
- 涉及频道覆盖时，先确认您改的是系统默认还是指定频道
- 如果只是临时排查问题，优先在测试频道执行
