# 用户管理

用户管理页面用来处理和用户本身有关的权限、状态和账号信息。

如果您要处理恶意用户、限制某个账号触发 AI，或者需要给某个账号单独开管理权限，通常都会在这里操作。

![用户管理列表页](/assets/advanced/user_management/user_list.png)

## 这个页面能做什么

- 设置用户禁止触发 AI
- 封禁用户
- 解封用户
- 重置密码（暂无作用，将于后续版本(>=2.3.3)做出调整）
- 查看用户状态和基础信息

## 状态说明

### 禁止触发

当用户被设置为禁止触发时，该用户将无法正常触发 AI 对话。  
但消息本身仍然可能进入上下文。

这个状态更适合：

- 其他机器人账号
- 其他 Nekro Agent 实例账号
- 您不希望它主动触发 AI，但又不想完全封掉的账号

### 封禁用户

当用户被设置为封禁时，该用户将无法继续使用 Nekro Agent，消息也会被直接忽略。

这个状态更适合：

- 恶意刷屏用户
- 滥用命令的用户
- 明确不希望继续交互的账号

### 解封用户

解封后，用户会恢复正常使用状态。

## 使用建议

![用户操作入口](/assets/advanced/user_management/user_actions.png)

- 对机器人账号，优先使用“禁止触发”
- 对恶意用户，直接封禁更干脆
- 涉及密码重置和高权限账号修改时，先确认目标用户是否选对
