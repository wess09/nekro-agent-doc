## More Possibilities

The application scenarios of Nekro Agent go far beyond these. With its powerful sandbox programming capabilities and plugin system, you can explore more innovative uses:

- **Learning Assistance Tools**: Generate practice questions, analyze difficult points, and provide personalized learning paths
- **Content Creation Assistant**: Assist in writing, generate creative ideas, and organize materials
- **Data Analysis Platform**: Integrated processing of data cleaning, statistical analysis, and result visualization
- **Smart Home Control**: Connect to smart home devices through API to achieve voice control

# Scenario Collection

If you have more interesting and interactive scenario cases, you are welcome to submit them through the following methods:

1. Click [Application Scenario Submission](https://github.com/KroMiose/nekro-agent-doc/issues/new?template=scenario_template.yml) to fill in and submit the template
2. When filling out, please try to describe the usage scenario, problems solved, and operation steps in detail
3. Screenshots or example dialogues will help us better understand your application scenarios
4. We will regularly organize high-quality scenarios into the documentation to share with everyone, and valuable scenario requirements may be supported by more specialized customized features in the future

Your creativity will help us explore more exciting interactive scenarios together! You are also welcome to join the [QQ Exchange Group](https://jq.qq.com/?_wv=1027&k=71t9iCT7) to exchange usage experiences with more users~
