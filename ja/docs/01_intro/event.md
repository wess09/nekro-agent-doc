# NekroAgent コミュニティプロモーションインセンティブプログラム 🚀

> **NekroAgent** は、マルチユーザーのクロスプラットフォームチャットボット向けの次世代オープンソースフレームワークです！ AIがコードを安全に実行できるだけでなく、QQ、Discord、Minecraft、Bilibili Liveなどのさまざまなプラットフォームもサポートしています。より多くの人にこのプロジェクトを知ってもらうために、このインセンティブプログラムを開始し、皆さんのプロモーションへの参加を心よりお待ちしています！

## 🎯 イベントの目標

このプログラムを通じて以下を達成したいと考えています：

- GitHubリポジトリの注目度と 🌟 を向上させる
- より多くの開発者、クリエイター、コミュニティユーザーの参加を引き付ける
- さまざまな技術プラットフォームへの影響力を拡大する
- より多くの本物のフィードバックを収集し、プロジェクトを継続的に改善する
- 開発チームのオープンソース活動に対する持続可能なモチベーションを提供する

> 💡 このイベントには **スポンサーはいません** - すべての報酬はプロジェクトチームメンバー自身によって資金提供されています
> 🙌 自発的なプロモーションや共有サポートも歓迎します

## 📝 参加方法

### ✅ 「参加意向登録フォーム」に記入する

> 📋 [NekroAgent プロモーションプログラム参加意向登録フォーム](https://docs.qq.com/form/page/DWXBFaFBtd2J2cHJZ) にアクセスして初期登録を行ってください

- 作成予定のコンテンツタイプ、プラットフォーム、公開予定時期などの情報を提出してください
- **インセンティブの対象となるには、2025年9月1日までに提出する必要があります**
- 以下の要件を満たす公開済みコンテンツについては、参加意向登録フォームを遡及して記入することで有効に参加できます

### ✍️ コンテンツを自主的に公開する

- 自由に記事を書いたり、動画を作成したりしてください
- 審査なしで自由に公開できます

### 📩 「コンテンツ登録フォーム」を提出する

> 📋 [NekroAgent プロモーションプログラムコンテンツ制作登録フォーム](https://docs.qq.com/form/page/DWXpFaUN5ZkpmRUVN) にアクセスしてコンテンツ制作登録情報を記入してください

- コンテンツが最初に公開された際に、公開アクセスリンクやその他の情報を登録フォームに登録してください
- **コンテンツ公開の7日後**に、フロントエンドおよびバックエンドデータのスクリーンショット、特別報酬条件、その他の情報で登録フォームを更新し続けてください
- 提出期限：**2025年10月31日まで**

**注：**

- プロジェクトチームの資金制限により、このイベントの意向登録は早期に終了する場合があります。参加に興味のあるユーザーは、できるだけ早くフォームに記入してください！

## 💰 報酬メカニズム

> ⚠️ すべての報酬計算は、**公開後7日目以降のデータスクリーンショット + 公開リンク** に基づいています
> 🚫 いかなる形式のデータ操作も禁止されています。発見された場合、資格は取り消されます。非操作目的の個人的なコミュニティ共有は許可されています

### 📌 インタラクション量について

**インタラクション量 = いいね + コメント + 弾幕 + お気に入り + 共有**
各プラットフォームのデータから **上位3つの次元を合計** し、プラットフォーム間の公平性を確保します

例：120いいね + 90弾幕 + 80お気に入り + 30コメント + 40共有 → 上位3つを取得 = 120 + 90 + 80 = **290**

### 📄 記事インセンティブ

- ✅ 基本報酬：50 RMB
- ➕ 100インタラクションごとに +10 RMB
- 🏅 プラットフォーム推奨（例：ホームページ/コラム） +100 RMB
- 💰 最大合計報酬：**300 RMB**

### 🎬 動画インセンティブ

#### ⏱ 長編動画（3分以上）

- ✅ 基本報酬：100 RMB
- ➕ 100インタラクションごとに +15 RMB
- ➕ 1000再生ごとに +20 RMB
- 💰 最大合計報酬：**700 RMB**

#### ⏱ ショート動画（1〜3分）

- ✅ 基本報酬：50 RMB
- ➕ 100インタラクションごとに +10 RMB
- ➕ 1000再生ごとに +10 RMB
- 💰 最大合計報酬：**300 RMB**

### ✅ 報酬計算例

> タイプ：Bilibili 長編動画
> 再生数：18,000 → 切り上げ +360 RMB
> インタラクション量（上位3つの合計）：420 → +60 RMB
> 最終報酬：100（基本）+ 360 + 60 = **520 RMB**

### 🏆 特別貢献報酬

以下の状況については、評価を行い、一回限りの報酬を提供します：

- 有名メディアによるコンテンツの転載
- GitHubへの大幅な成長またはコミュニティユーザーの流入をもたらした場合
- ニッチなコミュニティでの広範な普及と肯定的なレビュー

報酬内容には以下が含まれます：

- 👉 一回限りのボーナス上限：**1000 RMB**
- ✨ NekroAIリレー ステーション一回限りのAPI残高 100 USD
- 🏅 「プロモーション貢献者」となり、GitHubでの謝辞を受け取る

## ⚠️ 重要な注意事項

- コンテンツは本物で正確であり、誇張や捏造がないようにしてください
- 大部分または完全にAIによって自動生成された低品質なコンテンツは拒否されます
- リソースを参照する場合は、出典を引用し、他者の著作権を尊重してください
- **同一または非常に類似したコンテンツは1回のみ参加可能です**（例：動画 + 同じ作品のテキスト版は1つのエントリーと見なされ、最も高いデータに基づいて計算されます）
- 複数のプラットフォームで同時に公開されたコンテンツについては、最もボリュームの大きい単一プラットフォームのデータのみが1回計算されます
- 作成されたコンテンツは、関連するプラットフォームの規制および現地の法律や規制に準拠している必要があります
- コンテンツ公開後30日以内に達成された特別貢献報酬条件は有効と見なされます
- 作成されたコンテンツは公開され、アクセス可能である必要があります。プロジェクトチームメンバーが手動で審査し、報酬資格を確認します
- プロジェクトチームは最終解釈権を留保し、論争のあるコンテンツまたはこのイベントのルールに違反するコンテンツの参加資格を取り消す権利を有します
