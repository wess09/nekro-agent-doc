# ドキュメントサイトのプライバシーポリシー

最終更新日: 2025年12月16日

NekroAgent ドキュメントサイト（以下「本サイト」）へようこそ。私たちはあなたのプライバシーを非常に重視しており、個人情報の取り扱いについて透明性を確保することに取り組んでいます。このプライバシーポリシーは、私たちがどのように情報を収集、使用、保存、保護するかを理解していただくことを目的としています。

## 1. 情報収集

### 1.1 自動的に収集される情報
あなたが本サイトを訪問する際、以下の情報を自動的に収集する場合があります：
- Cookieデータ：閲覧体験を向上させるために使用されます

### 1.2 Cookieの使用
私たちは、Cookieおよび類似の技術を使用して以下のことを行います：
- 言語設定の記憶（VitePressの組み込みi18nはCookieに依存しています）
- ダークモードまたはライトモードの設定の記憶

ブラウザの組み込み機能を使用してCookieの設定を管理することで、Cookieを拒否することができます。

## 2. 情報の使用

収集した情報は、以下の目的で使用されます：
- ユーザー体験のパーソナライズ

## 3. 情報の共有

私たちはあなたの個人情報を第三者に販売することはありません。以下の状況下で情報を共有する場合があります：
- 法律および規制により要求される場合
