# ユーザー管理

## はじめに

ユーザー管理とは、ユーザーの権限、役割、情報を管理するプロセスです。Nekro Agentでは、ユーザー管理には以下が含まれます：

- ユーザーをAIトリガー禁止に設定する
- ユーザーを禁止する
- ユーザーの禁止を解除する

## ステータス説明

### トリガー禁止

ユーザーがトリガー禁止に設定されると、そのユーザーはAIとの会話をトリガーできなくなりますが、彼らのメッセージは依然としてAI対話コンテキストに入り、彼らに応答することができます。これは、他の非人間ユーザーや他のNAインスタンスアカウントを設定して、AIが互いに会話するデッドループに陥るのを避けるのに適しています。

### ユーザー禁止

ユーザーが禁止に設定されると、そのユーザーはNekro Agentのすべての機能を使用できなくなり、彼らの会話メッセージは完全に無視されます。これは、悪意のあるユーザーを設定するのに適しています。

### ユーザー禁止解除

ユーザーが禁止解除に設定されると、そのユーザーはNekro Agentのすべての機能を正常に使用できるようになります
