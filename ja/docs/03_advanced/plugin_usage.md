# プラグイン使用原則

:::warning

プラグインマーケットのプラグインは外部のプライベートリポジトリから来ています。プラグインのセキュリティ（コードレビューや権限評価を含む）はご自身で確認してください

:::

## プラグイン使用のベストプラクティス原則

### 機能の差別化

類似または重複する機能を持つ複数のプラグインを使用しないでください。これにより、AIコンテキストが冗長で混乱し、生成品質が低下する可能性があります
