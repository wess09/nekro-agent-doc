# プラグインエディタ

## プラグインエディタの紹介

プラグインエディタは、Nekro Agentのプラグインを編集するためのツールです。AI支援機能が統合されており、Nekro Agentプラグインの開発知識がプリセットされています。ユーザーのニーズに応じてプラグインコードを迅速に生成したり、簡単なプラグインを直接完成させたりすることができ、プラグイン開発効率を大幅に向上させます。

## プラグインエディタの簡単な使用方法

1.  WebUIの「プラグイン管理」で「プラグイン管理-プラグインエディタ」をクリックします
2.  新しいプラグインを作成するか、既存のプラグインを選択します
3.  プラグインエディタの横にある「プラグイン要件」に要件を入力します
4.  「AI生成」をクリックし、修正提案の生成を待ちます
5.  「エディタに適用」をクリックし、適用モデルが提案の適用を完了するのを待ちます
6.  「保存」をクリックしてプラグインを保存します

## プラグインの使用

プラグイン編集が完了したら、「プラグインを有効にする」をクリックしてプラグインを有効にできます。その後、プラグインが正常に読み込まれれば、「プラグイン管理」でプラグイン関連の情報を確認できます。

## AI能力モデル

エディタは、修正提案の生成と修正提案の適用という2つのAI能力を提供しています。「システム設定」-「基本設定」で、「プラグインコード生成モデルグループ」と「プラグインコード適用モデルグループ」という2つのモデルグループを見つけることができます。それらに適切な能力を持つモデルを設定してください。具体的に必要な能力は以下の通りです：

### プラグインコード生成モデルグループ

プラグインコード生成モデルグループは、プラグインコードを生成するために使用されます。コーディング能力と推論能力が高いモデル、例えば`gpt-4o`、`claude-3.7-sonnet`などを使用することが推奨されます。

### プラグインコード適用モデルグループ

プラグインコード適用モデルグループは、プラグインコードを適用するために使用されます。速度が速く指示遵守度が高いモデル、例えば`gemini-2.0-flash`などを使用することが推奨されます。
