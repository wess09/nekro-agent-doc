# ダッシュボード

ダッシュボードは、Nekro Agent の現在の稼働状態を素早く把握するための機能です。日常点検、異常にアクティブなチャンネルの調査、メッセージトレンドの確認、サービスが正常に動作しているかの確認に適しています。

::: tip まずダッシュボードを確認し、その後ログを見る
Bot が突然遅くなったり、静かになったり、呼び出し回数が異常になったりした場合は、まずダッシュボードで全体的なトレンドを確認し、その後ログセンターで詳細を確認することをお勧めします。
:::

## 確認できること

ダッシュボードには通常、以下の情報が表示されます：

| 領域 | 用途 |
|---|---|
| 概要カード | メッセージ、チャンネル、ユーザー、タスクなどの主要な数値を確認 |
| トレンドグラフ | 一定期間内のメッセージ量や呼び出し回数の変化を確認 |
| ランキングリスト | 最もアクティブなチャンネル、ユーザー、機能エントリを特定 |
| 分布統計 | 異なるメッセージタイプ、アダプター、状態の割合を確認 |
| リアルタイムステータス | サービスが現在オンラインかどうか、異常な変動があるかどうかを確認 |

バージョンによってページのレイアウトが若干異なる場合があります。WebUI の実際の表示に従ってください。

## よくある使用シナリオ

### Bot が突然応答しなくなった

以下の順序で確認できます：

1. ダッシュボードに新しいメッセージが入っているか
2. アクティブなチャンネルが特定のチャンネルに集中していないか
3. トレンドグラフがある時点から明らかに低下していないか
4. 同じ時間帯のエラーメッセージをログセンターで確認する

ダッシュボードに新しいメッセージがない場合は、まずアダプターの接続を確認してください。メッセージがあるが応答がない場合は、モデル、割り当て、ログを確認してください。

### 呼び出し回数が突然増加した

ランキングとトレンドに注目してください：

- 特定のチャンネルのメッセージ量が異常に高くないか
- 特定のユーザーが頻繁に Bot をトリガーしていないか
- 最近より緩いトリガー規則を有効化していないか
- 新しいプラグインやコマンドが追加の呼び出しをもたらしていないか

必要に応じて、チャンネル管理の無効化、監視モード、オーバーライド設定を使って範囲を制限できます。

### バージョン更新後の状態を確認する

Nekro Agent を更新した後、ダッシュボードで以下を確認できます：

- メッセージトレンドが正常に戻っているか
- チャンネルとユーザーのデータが正常に表示されているか
- 異常に高いエラーや失敗のトレンドが発生していないか
- ワークスペース、ナレッジベース、プラグインなどの機能にアクティビティがあるか

## データの更新

ダッシュボードのデータの一部は自動的に更新され、一部の統計は固定の時間ウィンドウで計算される場合があります。短時間でデータがすぐに変化しない場合は、少し待ってからページを更新してください。

ページが長時間更新されないが、Bot は実際に稼働している場合は、WebUI とバックエンドサービスの接続が正常かどうかを確認してください。
