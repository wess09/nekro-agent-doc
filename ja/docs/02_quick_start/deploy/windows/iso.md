#### Hyper-Vインポート手順
1. イメージをダウンロードして展開します
2. Hyper-Vマネージャーを開き、「仮想マシンのインポート」を選択します
3. 展開したフォルダを選択し、NekroAgentフォルダを選択します
4. 「仮想マシンをその場で登録（既存の一意のIDを使用）（R）」を選択します
5. 「次へ」を選択し、次に「完了」を選択します

:::warning 警告
インストール後すぐにシステムパスワードを変更してください

1. NapCatデフォルトパスワードを必ず変更してください：パブリックWebUIは強力なパスワードを使用する必要があります（12文字以上、数字、文字、記号を含むことを推奨）
2. トークン認証を必ず使用してください：OneBotサービスは有効なトークンを設定し、空のトークンや弱いパスワードを避けてください
3. セキュリティ上の理由から、デフォルトポートの使用は避けてください

:::
