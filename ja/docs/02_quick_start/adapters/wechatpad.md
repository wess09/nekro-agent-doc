## リスク説明

::: warning 高リスク
WeChatPad Pro を含むすべての Bot フレームワークに、普段使いの WeChat アカウントを接続することは推奨されません。アカウント異常、機能制限、最悪の場合は BAN のリスクがあります。
:::

特に次の点に注意してください:

- API の実行環境とスマートフォンの所在地が大きく離れていると、リスク制御が発動しやすい
- リモートのクラウドサーバーからのログインはリスクが高い
- 手動操作がないまま長時間かつ高頻度で自動化すると、アカウント異常を起こしやすい
