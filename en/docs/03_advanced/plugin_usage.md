# Plugin Usage Principles

:::warning

Plugins in the plugin market come from external private repositories. Please check the security of plugins yourself, including code review and permission assessment

:::

## Best Practice Principles for Plugin Usage

### Functional Differentiation

Avoid using multiple plugins with similar or duplicate functions, as this may lead to redundant and confusing AI contexts, resulting in decreased generation quality
