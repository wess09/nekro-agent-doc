# Session Independent Persona

## Introduction

Nekro Agent supports using independent personas for each session. You can set different personas for each session to achieve better interaction effects.

## Application Scenarios

1. **Multi-role Conversations**: Have conversations with multiple different AI roles simultaneously, with each session maintaining an independent personality
2. **Domain Differentiation**: Set different personas for entertainment groups and productivity groups, balancing work and entertainment!
3. **Testing and Comparison**: Test the response effects of different personas in parallel

## How to Set Session Independent Persona

1. **Add Persona Information**

   - Go to "Persona Management" in WebUI
   - Click the "Create New Persona" button
   - Fill in basic information such as persona name, description, characteristics, etc.
   - Set the default behavior and response style of the persona
   - Save the persona

2. **Select Session Persona**

   - Select the target session in "Session Management"
   - Click the "Set Persona" button
   - Select a pre-configured persona from the list
   - Confirm the selection
