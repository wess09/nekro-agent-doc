# WeChat (WeChatPad Pro) Configuration Guide

::: danger Warning

This adapter guide is not finished. Nekro Agent does not recommend using this adapter to connect WeChat.

:::

## Risk Notes

::: warning High Risk
It is not recommended to use any primary WeChat account with any bot framework, including WeChatPad Pro. There is a risk of account abnormalities, feature restrictions, or even account bans.
:::

You should also pay special attention to the following:

- If the API runtime environment is far away from the location of your phone, risk control is easier to trigger
- Logging in from a remote cloud server has a higher risk
- Long-running high-frequency automation without manual activity can also trigger account abnormalities
