## <DocIcon name="rocket" /> Core Capabilities

NekroAgent guides AI to generate accurate code and execute it in a sandbox through a powerful and flexible prompt construction system, interacting with the real environment via RPC communication. Key features include:

- Secure containerized **sandbox execution environment** with a complete Python runtime environment
- Powerful code **guidance, generation, error correction, and execution** capabilities, supporting complex tasks such as chart generation and general content processing
- Native multimodal **visual understanding and interaction** support, capable of processing images, files, and other resources
- Highly extensible **plugin system** providing various key node callbacks, prompt injection, and custom sandbox methods
- Native **multi-user scenario interaction** support, efficiently understanding group chat scenario requirements
- Cloud-based real-time resource sharing services, including plugins, personas, and other powerful and friendly **community-driven** capabilities
- Feature-rich **visual interface** application management panel
- Rejection of invalid prompts and abuse of iterative agents, achieving ultimate **cost reduction and efficiency improvement** for complex tasks
- **Event-driven** interaction architecture following an async-first efficient response mechanism
- Support for dynamic installation and **real-time introduction of PYPI dependencies**, unlocking the endless treasures of the Python universe

## <DocIcon name="star" /> Nekro AI

**Nekro AI** is the cloud service platform of Nekro Agent, serving as the service hub of the entire ecosystem, providing unified management and distribution capabilities for plugins and personas. It is committed to helping users and developers build and share intelligent AI agent systems more efficiently.

- **Plugin Store**: Centralized management of various functional extension plugins, supporting one-click installation, upgrades, and graphical configuration, facilitating developers to publish and share capability modules
- **Persona Store**: Provides a diverse character persona sharing platform, supporting upload, download, and personalized customization to meet agent personality construction needs in various scenarios

Nekro AI is committed to creating an open, collaborative, and highly available service ecosystem, promoting the significant development of the Nekro Agent application ecosystem!
