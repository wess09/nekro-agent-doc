# NekroAgent Community Promotion Incentive Program <DocIcon name="rocket" />

> **NekroAgent** is a next-generation open-source framework for multi-user cross-platform chatbots! It not only enables AI to safely execute code but also supports various platforms such as QQ, Discord, Minecraft, and Bilibili Live. To help more people discover this project, we've launched this incentive program, sincerely inviting everyone to promote it together!

## <DocIcon name="target" /> Event Goals

We hope to achieve the following through this program:

- Increase GitHub repository attention and <DocIcon name="star" />
- Attract more developers, creators, and community users to join
- Expand influence across various technical platforms
- Collect more genuine feedback to continuously improve the project
- Provide sustainable motivation for the development team's open-source efforts

> <DocIcon name="lightbulb" /> This event has **no sponsors** - all rewards are funded by project team members themselves
> <DocIcon name="hand" /> We also welcome voluntary promotion and spontaneous sharing support

## <DocIcon name="circle-dollar" /> Reward Mechanism

> <DocIcon name="warning" label="注意" /> All reward calculations are based on **data screenshots + public links from the 7th day after publication**
> <DocIcon name="ban" /> Any form of data manipulation is prohibited. If discovered, eligibility will be revoked. Personal community sharing for non-manipulation purposes is allowed

### <DocIcon name="target" /> About Interaction Volume

**Interaction Volume = Likes + Comments + Danmaku + Favorites + Shares**
We will **sum the top three dimensions** from each platform's data to ensure fairness across platforms

For example: 120 likes + 90 danmaku + 80 favorites + 30 comments + 40 shares → take top three = 120 + 90 + 80 = **290**

## <DocIcon name="warning" label="注意" /> Important Notes

- Content must be authentic and accurate, without exaggeration or fabrication
- Reject low-quality content that is largely or entirely auto-generated by AI
- If referencing resources, please cite the sources and respect others' copyrights
- **The same or highly similar content is only allowed to participate once** (e.g., video + text version of the same work is considered one entry, calculated based on the highest data)
- For content published on multiple platforms simultaneously, only the data from the single platform with the highest volume will be calculated once
- Created content must comply with relevant platform regulations and local laws and regulations
- Special contribution reward conditions achieved within 30 days after content publication are considered valid
- Created content needs to be public and accessible. Project team members will manually review to confirm reward eligibility
- The project team reserves the right of final interpretation and has the right to cancel participation eligibility for controversial content or content that violates the rules of this event
